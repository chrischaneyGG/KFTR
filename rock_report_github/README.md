## How to update the Seiler logo

1. Open `index.html` in a text editor
2. Find the comment `<!-- Replace .seiler-logo-box with <img> -->`
3. Replace the `<div class="seiler-logo-box">...</div>` block with:
   ```html
   <img src="seiler-logo.png" alt="The Seiler Group of Raymond James" style="height:60px;" />
   ```
4. Add the `seiler-logo.png` file to this repo
5. Commit and push — GitHub Pages updates in ~60 seconds

---

## Deploying to GitHub Pages

1. Push this repo to GitHub (must be **Public**)
2. Go to **Settings → Pages**
3. Under *Source*, select **Deploy from a branch**
4. Choose **main** branch → **/ (root)** folder → **Save**
5. Your URL: `https://[your-username].github.io/[repo-name]/`
